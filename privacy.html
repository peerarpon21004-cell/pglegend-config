PGLEGEND — Privacy Policy

Last updated: 13 May 2026

สรุปสั้นๆ

PGLEGEND ไม่เก็บข้อมูลส่วนตัวของคุณ ไม่มีการสมัครสมาชิก ไม่มี analytics ไม่มี tracking ทุกอย่างอยู่ในเครื่องของคุณ

1. ข้อมูลที่เก็บ

ไม่เก็บข้อมูลใดๆ ที่ผูกกับตัวตนของคุณ

ข้อมูลในเครื่อง (Local Only)

  • Progress การเล่น (level, coin, skill ที่ปลด, trophy)
  • Settings (เปิด/ปิดเสียง, vibration)
  • Daily Quest progress
  • Shop purchases (cosmetic frame, boost)

ทั้งหมดเก็บใน Documents/pglegend_save.json ในเครื่องคุณเท่านั้น ไม่ส่งไปที่ไหน — ลบแอพ = ข้อมูลทั้งหมดถูกลบ

2. การติดต่อภายนอก

แอพเชื่อมต่ออินเทอร์เน็ตเพียงครั้งเดียวต่อการเปิดแอพ:

  • GitHub Pages (peerarpon21004-cell.github.io) — ดึงไฟล์ JSON config เพื่อตรวจสอบว่าแอพอยู่ในโหมดปรับปรุงหรือไม่

คำขอนี้ไม่ส่งข้อมูลส่วนตัวใดๆ — เป็นแค่ HTTP GET ธรรมดา

3. Cookies และ Tracking

ไม่มี ไม่ใช้ third-party SDK ไม่ติดตามผู้ใช้ ไม่มี advertising

4. การซื้อในแอพ (In-App Purchase)

เฟส 1 launch ยังไม่มี IAP จริง coin ที่ใช้ได้ทั้งหมดมาจากการเล่นเกม หากภายหลังเพิ่ม IAP จะอัปเดต Privacy Policy นี้ก่อนเปิดใช้

5. เด็กและเยาวชน (COPPA)

เกมเหมาะสำหรับอายุ 9+ เนื่องจากเกมไม่เก็บข้อมูลใดๆ จึงปฏิบัติตาม COPPA โดยอัตโนมัติ

6. การเปลี่ยนแปลงนโยบาย

หากนโยบายมีการเปลี่ยนแปลง ผู้ใช้จะเห็น banner แจ้งเตือนในแอพ Last updated date จะอัปเดตด้านบนสุด

7. ติดต่อ

หากมีคำถาม ติดต่อ: peerarpon21004@icloud.com

TL;DR

PGLEGEND does not collect any personal data. No account required, no analytics, no tracking. Everything stays on your device.

1. Data Collected

No personally identifiable information is collected.

On-Device Only

  • Game progress (level, coins, unlocked skills, trophies)
  • Settings (audio mute, haptics on/off)
  • Daily quest progress
  • Shop purchases (cosmetic frame, boost)

All data is stored in Documents/pglegend_save.json on your device only. Nothing is transmitted. Deleting the app removes all data.

2. Network Activity

The app makes exactly one outbound HTTPS request per cold launch:

  • GitHub Pages (peerarpon21004-cell.github.io) — fetches a JSON configuration file to determine maintenance state.

This request does not transmit any personal information — it is a plain HTTP GET.

3. Cookies and Tracking

None. No third-party SDKs. No user tracking. No advertising.

4. In-App Purchases

Phase 1 launch contains no real-money IAP. All in-game currency is earned by playing. Any future IAP additions will trigger an update to this policy.

5. Children's Privacy (COPPA)

The game is appropriate for ages 9+. Because no data is collected, the app complies with COPPA by design.

6. Changes to this Policy

If this policy changes, the app will display an in-app banner notifying users. The "Last updated" date above will reflect any revision.

Questions: peerarpon21004@icloud.com
